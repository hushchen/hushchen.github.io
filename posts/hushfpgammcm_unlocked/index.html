MMCM Unlocked - Vivado Simulation

Posted on Aug 26, 2023

In a project, the target component is Xilinx Ultrascale Kintex series. We do the RTL simulation using Vivado. I encountered the MMCM can’t be locked situation seval times.

Sometimes I forgot how to solve this problem that spend me some hours to found the root cause. For the reason, I record this issue here.

There is a IP compoent used in the project - STARTUP 3. Because we design a on-line update configuration function that will be integrated with the STARTUP 3 component. Actually the update configuration function works well on the board. When we do the simulation, the MMCM is unlocked all the time. The main reason is the STARTUP 3 component. We just need to simulate the others functions. We use a SIMULATION_FLAG to mask the STARTUP 3 component when simulation.
