Github Pages 快速架設網站

Posted on Jan 1, 0001

Github Pages提供了一個快速便利的靜態網頁空間, 非常適合架設個人網站或是部落格, 以下方法介壽如何利用Github Pages架設網站.

首先,先在Github註冊一個帳號, 並在這個帳號底下建立一個Repositor,這個Repositor是用來管理Github Pages的內容,

需要注意Repositor命名方式,以註冊帳號為命名:name.github.io

接下來準備一個Hello Word的index.html檔案,html內容如何編寫可以在網路上找到範例, 完成後將該index.html放入Repositor中. 並在此Repositor中編寫readme.md檔案,該檔案是用於github中的說明,與網站內容無關.

以上完成後,開啟網頁https://yourname.github.io/,可以看到Hellow Word網頁, 網頁架設即完成囉.
